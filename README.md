# Rock-Paper-Scissors
Rock–Paper–Scissors Game in JavaScript

# Description
This is a simple Rock–Paper–Scissors game built using JavaScript.
The game allows a human player to play against the computer for 5 rounds. After all rounds are played, the winner is determined based on the scores.

# Features

- Human vs Computer gameplay

- Random computer moves

- Input is case-insensitive and ignores extra spaces

- Keeps track of scores per round

- Displays final winner after 5 rounds

- Simple console-based interface using prompt() and console.log()

# Example Run

Choice between 'rock' or 'paper' or 'scissors': rock
Computer chose scissors
Win
Round 1: Human 1 - Computer 0

Choice between 'rock' or 'paper' or 'scissors': paper
Computer chose rock
Win
Round 2: Human 2 - Computer 0

...
the winner is human

# Installation

- Save the JavaScript file (e.g., game.js) to your computer.

- Open it in any modern web browser.

- Open the developer console to see game output (or run in an environment that supports prompt()).
